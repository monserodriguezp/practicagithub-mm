Pasos para trabajar con git

Una vez que mi compañera crea la carpeta y la sube a github, yo tengo que hacer click derecho en alguna carpeta de mi ordenador (escritorio por ejemplo) y seleccionar "Git bash here". Al abrirse la consola, ejecuto el comando clone con la url proporcionada en la plataforma de git hub para descargar de forma local todos los archivos. De esta forma, ya se puede trabajar y realizar las tareas necesarias.

En primer lugar, ejecutamos comando cd [nombre-carpeta] para ir a la raiz de la carpeta. A continuacion, git checkout -b [nombre-rama], git add . para añadir nuestros cambios, git commit -m [comentario-tarea], git push [copiamos url], de nuevo git push con la url indicada, introducimos nuestras credenciales y se subirá todo a nuestro directorio remoto en github

Actualizamos nuestra ventana y veremos una nueva rama añadida. Al entrar, pulsamos en el boton "new pull-request", una vez dentro, seleccionamos el botón "create pull-request" que nos llevará a otra página. Habrá que copiar la url para pasársela al compañero. Cuando el compañero entra, tiene que entrar en "file changes", ver los cambios, comentar si fuese necesario y aprobarlos. Al hacer todo esto, mergeamos todo el flujo para que se fusionen.
